#  Authentication and Authorization App in SwiftUI

This is a simple SwiftUI application that demonstrates user authentication and authorization. The app allows users to sign up, log in, and access protected content based on their authentication status. This app was build as a practice project to learn SwiftUI and security authentication concepts using Go as the backend.
